  1. 次のソースプログラムは計算式を含んでいますが、 実行しても計算結果を知ることができません。 

    /*
** プログラム名: kadai3
** 動作: 計算式を含むプログラム
** 作者: 自分の名前、学籍番号
** 日付: 今日の日付
*/
main()
{
    (1 + 2) * 4;
    return 0;
}

  2. なんらかの方法で計算結果を知ることができるよう、 プログラムを修正してください。 修正したソースプログラムのファイル名は kadai3.c としてください。

  3. kadai3.c をメールで tokoiまで送ってください。 Subject: は kadai3 としてください。
