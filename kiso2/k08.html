課題８

  1. 次のソースプログラムを kadai8.c というファイル名で作成してください。 

    /*
** プログラム名: kadai8
** 動作: ３×５を計算し、その２倍および３倍を出力する
** 作者: 自分の名前、学籍番号
** 日付: 今日の日付
*/
int main(void)
{
    int x;

    x = 3 * 5;
    printf("%dの２倍は%d", x, x * 2);
    printf("%dの３倍は%d", x, x * 3);
    return 0; /* 正常終了 */
}

  2. このソースプログラムをコンパイル／リンクし、 kadai8 という名前の実行プログラムを作成してください。

  3. kadai8 を実行してください。

  4. kadai8 を実行すると、 
    「〜の２倍は〜〜の３倍は〜」
    という具合に、 出力が１行で表示されると思います （プロンプトの % も同じ行に現れるかも知れません）。 これを、 
    「〜の２倍は〜」
「〜の３倍は〜」
    のように、２行に分けて出力するよう kadai8.c を修正し、 それをメールで tokoiまで送ってください。 Subject: は kadai8 としてください。
