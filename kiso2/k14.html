課題１４

  1. 次のソースプログラムを kadai14.c というファイル名で作成してください。 その際 c には自分の名前のイニシャルを、 n には名前のイニシャルと名字のイニシャルの文字コードの差を入れて、 プログラムを実行したとき 自分のイニシャルが正しく表示されるようにしてください

  2. 例えばイニシャルが BA なら、 c は B、 A の文字コードは B の１字前にあるので n は -1 になります。 

    /*
** プログラム名: kadai14
** 動作: 整数と文字の計算
** 作者: 自分の名前、学籍番号
** 日付: 今日の日付
*/
int main(void)
{
    char first; /* 名前のイニシャル */
    char last;  /* 名字のイニシャル */

    first = 'c';
    last = first + (n);
    printf("私のイニシャルは %c%c です\n", first, last);
    return 0;
}

  3. このソースプログラムをコンパイル／リンクし、 kadai14 という名前の実行プログラムを作成してください。

  4. kadai14 を実行し、 正しくイニシャルが表示されたら、 kadai14.c をメールで tokoiまで送ってください。 Subject: は kadai14 としてください。
