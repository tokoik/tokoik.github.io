課題２

  1. kadai1.c を kadai2.c にコピーしてください。 ファイルのコピーは次のようにして行います。 
    % cp kadai1.c kadai2.c
  2. kadai2.c の return の前に、 次の２行を挿入してください。 
    printf("Hello! ");
printf("World\n");
  3. kadak2.c をコンパイル／リンクし、 kadai2 という名前の実行プログラムを作成してください。

  4. kadai2 を実行し、表示されるものを確かめてください。

  5. 次に、kadai2.c の２つの printf が、 １行になるように修正してください。 
    printf("Hello! ");printf("World\n");
    これは、
    • ２つ目の printf の行の先頭にカーソルを移動し、Backspace をタイプ
    • １つ目の printf の行の行末にカーソルを移動し、C-k をタイプ
    のいずれかで行うことができます。

  6. 再び kadak2.c をコンパイル／リンクし、 kadai2 という名前の実行プログラムを作成してください。

  7. kadai2 を実行し、 結果が先程と変わらないことを確かめてください。

  8. 以上の作業が終了したら、 kadai2.c を tokoi までメールで送ってください。 このとき Subject: は kadai2 にしてください。
