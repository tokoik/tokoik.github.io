  1. 次のプログラムは、 入力した名前を素のまま表示するだけのプログラムです。 

    /*
** プログラム名: kadai23
** 動作: 文字列の入力
** 作者: 自分の名前、学籍番号
** 日付: 今日の日付
*/
#include <stdio.h>

int main(void)
{
    char name[100];

    printf("あなたの名前をいれてください:");
    scanf("%99s", name);
    printf("あなたの名前は %s です\n", name);

    return 0;
}

  2. このプログラムは "Horosue Ryoko" のように、 名前の間に空白があると、 空白の前の部分しか表示しません。

  3. このプログラムを、 １個の空白をはさんで姓名を入力したときに、 それが全て表示されるように修正してください。

  4. ソースプログラムのファイル名は kadai23.c とし、 それをメールで tokoiまで送ってください。 Subject: は kadai23 としてください。
