課題１９

  1. 次のソースプログラムをもとにして、 自分の名前を "My name is ????" というように出力するプログラムを 作ってください。

  2. 配列変数 name の要素の数などを 自分の名前の文字数に合わせることを忘れないようにしてください。

  3. ソースプログラムのファイル名は kadai19.c としてください。 

    /*
** プログラム名: kadai19
** 動作: 自己紹介
** 作者: 自分の名前、学籍番号
** 日付: 今日の日付
*/
#include <stdio.h>

int main(void)
{
    char name[4]; /* 名前を入れる変数（要素数を指定）*/

    name[0] = 'K';
    name[1] = 'e';
    name[2] = 'n';
    name[3] = '\0';

    printf("My name is %s.\n", name);

    return 0;
}

  4. このソースプログラムをコンパイル／リンクし、 kadai19 という名前の実行プログラムを作成してください。

  5. kadai19 を実行し期待した出力が得られたら、 ソースプログラム（kadai19.c）をメールで tokoiまで送ってください。 Subject: は kadai19 としてください。
