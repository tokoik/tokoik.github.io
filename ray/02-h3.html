球と視線の交差

球の方程式

(P - Pc)2 = r2

に視線の方程式

P = Vt + Pe

を代入すれば，

(Vt + Pe - Pc)2 = r2

これを展開して整理すれば，

V2t2 + 2V・(Pe - Pc)t + (Pe - Pc)2 - r2 = 0

したがって，At2 + 2Bt + C = 0 の係数 A, B, C は，

A = V2 = vx2 + vy2 + vz2
B = V・(Pe - Pc) = vx * (ex - cx) + vy * (ey - cy) + vz * (ez - cz)
C = (Pe - Pc)2 - r2 = (ex - cx)2 + (ey - cy)2 + (ez - cz)2 - r2
