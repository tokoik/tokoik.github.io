拡散反射成光（＋環境反射光）の強度の求め方

この場合の反射光強度 I は，次式で求めることができる．ここで N および L は単位ベクトルである必要がある．

rd = N・L
Id = rd Il
I = Kd Id + Ka Ia

I を求める手順は以下のようになる．

  1. 視線と物体との交点が見つかった時点で， その交点における法線単位ベクトル N を求める．
  2. N と L の内積 rd = N・L を求める．
  3. rd < 0 なら，その点には光源からの光が当らない． rd = 0 の場合もこの光源による反射光がないわけだから， これらの場合，反射光は環境光によるものだけになる． したがって，
    もし rd > 0 なら
    Id = rd Il
    それ以外
    Id = 0
    として，I = Kd Id + Ka Ia を求める．
