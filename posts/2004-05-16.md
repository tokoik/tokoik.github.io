## (1) /etc/userdb にユーザのエントリを作成する

<pre>
# /usr/sbin/userdb <em>foo</em> set home=<em>/home/foo</em> mail=<em>/home/foo/Maildir</em> uid=<em>1234</em> gid=<em>1234</em>
```


## (2) 作成したエントリに login 認証用のパスワードを追加する

<pre>
# /usr/sbin/userdbpw | /usr/sbin/userdb <em>foo</em> set imappw
Password: 
Reenter password: 
```


## (3) 作成したエントリに CRAM-MD5 認証用のパスワードを追加する

<pre>
# /usr/sbin/userdbpw -hmac-md5 | /usr/sbin/userdb <em>foo</em> set hmac-md5pw
Password: 
Reenter Password: 
```


## (4) /etc/userdb のデータベースを更新する

<pre>
# /usr/sbin/makeuserdb
```


## (5) /etc/courier-imap/imapd を変更する

<pre>
IMAP_CAPABILITY="IMAP4rev1 UIDPLUS CHILDREN NAMESPACE THREAD=ORDEREDSUBJECT
THREAD=REFERENCES SORT QUOTA IDLE"
↓
IMAP_CAPABILITY="IMAP4rev1 UIDPLUS CHILDREN NAMESPACE THREAD=ORDEREDSUBJECT
THREAD=REFERENCES SORT QUOTA IDLE <em>AUTH=CRAM-MD5 AUTH=CRAM-SHA1 AUTH=LOGIN</em>"
```


## (6) courier-imapd を再起動する

<pre>
# /etc/init.d/courier-imap restart
```
