## GLFW3 を Linuxmint にインストールする

GLFW3 をソースからビルドして [Linuxmint](http://www.linuxmint.com/) にインストールしたときの手順をメモっておきます. 多分, [Ubuntu](http://www.ubuntu.com/) でも同じジャナイカと思います. 既にコンパイラなどの基本的な開発環境がインストールされているものとします.
まず GLFW のソースパッケージを, [GLFW プロジェクト](http://www.glfw.org/) のトップページの &quot;Download GLFW 3.1.1&quot; のボタンか, [Download](http://www.glfw.org/download.html) のページの &quot;Source Package&quot; のボタンをクリックして入手します. それを以下の手順でインストールします.

<ol>
<li>[CMake](http://www.cmake.org/) をインストールします.
```
<li>XRandR, Xinerama, Xcursor, および X11 Input Extension Library (Xi) をインストールします.
```
<li>ダウンロードしたソースパッケージを適当なところで展開します. 
```
<li>展開したディレクトリ内に build というディレクトリを作成します.
```
<li>作成したディレクトリに cd します.
```
<li>そこで cmake します.
```
<li>cmake に成功したら make します.
```
<li>make に成功したら make install でインストールします.
```
</ol>
