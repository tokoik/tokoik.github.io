補足

chaostoy.c, lorenz1.c, lorenz2.c, rossler.c は柴山先生によるサンプルプログラムです． そのほかのプログラムは床井が勝手に立体視などの手を加えたものです． 立体視版のうち，make して *-crm というファイル名になったもの (-DCRM を付けてコンパイルされたもの) は OpenGL の Quad Buffer Stereo を使用しています．それ以外のものは SGI のマシン（Indy や O2 等）でしか立体視できません．
