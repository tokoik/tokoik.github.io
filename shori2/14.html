１．ファイル操作（データ処理）

たとえば次のような内容のテキストファイル tabemono があったとします。

tabemono 
taiyaki   100 24
takoyaki  300  8
okonomi   400  7

これはあるスーパーの入り口に出ている屋台の、 その日の売り上げ記録だったとします。 この一つ目の欄は商品名、２つ目は単価、３つ目を個数です。 このデータから、 この屋台のその日の売り上げ合計を求めるプログラムを作ります。

yatai.cc 
#include <fstream.h>

int main(void) {
  ifstream data("tabemono");
  char shouhin[10];
  double tanka, sum = 0.0;
  int kosuu;

  while (data >> shouhin >> tanka >> kosuu) {
    sum += tanka * kosuu;
  }

  cout << "合計は " << sum << "円" << endl;
}
  • ファイル tabemono を ifstream クラスのオブジェクト data を作成して開きます。
  • data >> shouhin >> tanka >> kosuu で１行文のデータを読み出します。
  • 上のように >> を連続して指定できるのは、 data >> shouhin という計算の「値（答）」が data というオブジェクトそのものだからです。
  • 抽出演算子 >> による処理が入力データの終りやエラーなどで 「失敗」すると、この計算の値は 0（ヌルポインタ）になります。
  • したがって、上のプログラムではデータを最後まで読み出すと、 while () による繰り返しから抜け出します。

このプログラムには一つ問題があります。 配列 shouhin のサイズは 10 なので、 入力データに含まれる商品名の文字数は９文字までに制限する必要があります。 ここではとりあえず入力データの商品名は９文字以内として考えます。
