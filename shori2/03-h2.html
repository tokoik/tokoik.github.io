補足：終了ステータスについて

終了ステータスは、 プログラムが正常に実行できたか否かなどの情報を、 プログラムからオペレーティングシステムに伝えるための手段です。

  • 例えば csh（% を表示して利用者の指令～コマンドを待っているプログラム） の && という演算子は、 終了ステータスを参照して「条件判断」のような動作をします。 これは次のような形で使われます。 

    % command1 && command2

    このとき command2 は、command1 の終了ステータスが 0 の時、 すなわち command1 が「正常終了」したときに限って実行されます。

  • 終了ステータスは プログラムになんらかのエラーが起きた場合に限らず、 プログラム自身の処理結果をオペレーティングシステムに 伝えたいときにも使われます。 その典型的なものが test コマンドでしょう。 

    % test 1 = 1 && echo true
true              ← echo true が実行された
% test 1 = 2 && echo true
%                 ← echo true が実行されなかった

  • プログラムを実行すると、 オペレーティングシステムは main() 関数を呼び出し、 main() 関数の戻り値を終了ステータスに用います。 プログラムが正常に終了した場合、 プログラムは終了ステータスとして通常 0 を戻します。 終了ステータスは整数値なので、 main() 関数のデータ型には int を用います。

    int main(void) {
  return 0;
}

  • プログラム中の任意の位置でプログラムを終了させたいときは、 exit() という ライブラリ関数を用います。 この場合は終了ステータスを引数で指定します。 下のプログラムでは main() 関数の戻り値は 0 ですが、 sub() の実行中に exit(1) でプログラムが終了するため、 main() において sub() よりあとの処理は実行されません。 また、このとき exit(1) の引数 1 が終了ステータスになります。 なお exit() を使用するときは、 それを使っている場所より前に #include <stdlib.h> を置いておいてください。 

    #include <stdlib.h>

void sub(void) {
  ...
  exit(1);
}

int main(void) {
  ...
  sub();
  ...

  return 0;
}

  • UNIX に用意されている様々なコマンドのうち ほとんどのものは、 終了ステータスになんらかの意味を持っています。 これをうまく使えば、 複数のコマンドを組み合わせて結構凝った仕事が簡単に実現できます。
